## 简单概述
这是我的毕业设计，使用H5 Runtime + Mui + HBuilder结合开发

## 核心功能
1. 多人定位、导航；
2. 与好友聊天、群聊、添加好友、添加群；
3. 登陆注册，支持第三方登陆（QQ、微信、微博）；

## 技术实现
- 开发工具： **HBuilder**
- 数据请求： **Ajax**
- 页面渲染： **Vue**
- UI 框 架： **Mui**
- 后    台： **Java**
